# nicxlive D→C++ 移植フロー

以下の手順で、D 言語版 `nijilive` の実装と C++ 版 `nicxlive` の実装差分を埋め、完全互換を目指す。

1. **変換対象の明確化**  
   - D 言語から変換すべきファイル一覧を `doc/tasks.md` に進捗管理できる形式（例: チェックボックス付きリストやステータス列付き表）で記載する。
2. **写経ベースの変換**  
   - D 言語実装を正として 1:1 で C++ に置き換える。ロジック・フィールド構成・シグネチャを忠実に移植することを基本とする。すでにC++に実装がある場合、D言語との差分を抽出して、それを１：１で実装する。
3. **メソッド/フィールド単位の差分検証**  
   - 生成した C++ と元の D 実装の差異をメソッド・フィールドレベルで列挙し（グルーピングしない）、批判的に検知・記録する。  
   - 各項目に評価値（◯ / △ / ✗）を付け、互換性の達成度を可視化する。
4. **全項目が◯になるまで反復**  
   - 評価に ✗ や △ が残る限り、2〜3 を繰り返し修正を加え、全項目が ◯ になるまで継続する。

このサイクルを全対象ファイルで適用し、`doc/tasks.md` を最新状態に保つこと。進捗と検証結果は常にドキュメント化し、再現性を確保する。
